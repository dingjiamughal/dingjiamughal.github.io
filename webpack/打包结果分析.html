# 分析打包结果

# 官方方案

webpack --profile --json | Out-file 'stats.json' -Encoding OEM

将打包结果的json文件上传到 http://webpack.github.io/analyse

#webpack-bundle-analyzer 可视化查看打包结果

cnpm install webpack-bundle-analyzer --save-dev

在 GitHub 上编辑此页
